A interactive console-based Todo app built with C# and .NET.

Navigate with arrow keys, manage tasks easily, and enjoy a simple UI

✨ Features

✅ Add, edit, delete todos  
🎮 Arrow-key navigation  
🔄 Toggle task status  
🔍 Search  
📊 Filter & sort  
🚦 Priority system  
⏰ Overdue detection  
💾 Saves data using JSON  



🎮 Controls

| Key | Action |
|-----|--------|
| ⬆️ / ⬇️ | Navigate |
| ⏎ | Select / toggle |
| E | Edit |
| D | Delete |
| ESC | Back |


🚀 Run the app

```bash
dotnet run
